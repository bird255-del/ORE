Overview
This program was created to provide a simple, customizable reminder tool for Microsoft Outlook users who want to stay organized and on top of their tasks.

When you're working on a project or have another window maximized, the Outlook Reminder window may appear behind your work—making it easy to miss important events.
This tool ensures you never overlook an Outlook reminder again by detecting when a reminder window appears and triggering a visual alert.

Important Note
This program detects any window with the word "(reminder)" in its title.
If you use other applications that include “reminder” in their window titles, they may also trigger the alert. Please keep this in mind while using the tool.
